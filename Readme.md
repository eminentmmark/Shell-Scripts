to make the (text) file executable run 
 
 	chmod 755 <file_name.py>
 	

to exectue the file run

	./<file_name.py>
	
	
to create a file

	touch <file_name>.filetype
	
	
rename all .txt files to .bak

	mv *.txt *.bak

	cp /tmp/a/* /tmp/b/
	cp /tmp/a/*.txt /tmp/b/
	cp /tmp/a/*.html /tmp/b/
